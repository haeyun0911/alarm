## 설치
- pip install opencv-python
- pip install ultralytics
- pip install mediapipe
- pip install pillow

## 실행
1. 캠버전 python test5.py
2. 저장된 영상 버전 python test5_1.py
- video_path 변수에 알맞는 영상 경로 입력 필수

## 문제해결 FAQ
1. numpy 버전 충돌시 
- pip uninstall numpy
- pip install numpy==1.26.4

2. 한글이 출력되지 않고 깨져보임
- 한글 폰트 경로를 올바르게 입력했나 확인
